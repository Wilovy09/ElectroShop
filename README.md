## Comandos

```sh
cargo install sqlx-cli --no-default-features --features sqlite
sqlx migrate add create_<ENTITIE>s_table
# Para esto ya tiene que estar creada la base de datos aunque sea vacia.
sqlx migrate run
cargo watch -x run
```
